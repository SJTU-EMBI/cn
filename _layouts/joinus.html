加入我们

第二届 SJTU AIGC Class 已经在2024年11月17日截止招募。

创立背景

从多年前的Sobel与Canny的一众数字图像处理算法，帮助我们检测图像中的边缘。

再到R-CNN到YOLO深度学习算法，助力图片中的目标检测。

从对抗生成模型(GAN)到扩散模型(Diffusion Model)，从二维平面到三维空间，深度学习模型展示着无限量的内容生成的可能性。

从ChatGPT到文心一言，从MidJourney到DALL·E，人工智能生成内容的应用快速的铺展开来。

人工智能生成内容（Artificial Intelligence Generated Content, AIGC）已经逐渐靠近我们的生活，成为扩展现实的重要发展领域，也必将会成为媒体、文化和创意产业的关键驱动力。

2023年秋季，上海交通大学学生创新中心、上海交通大学人工智能研究院联合华为技术有限公司、联想集团与上海莉莉丝科技股份有限公司创立“上海交通大学 AIGC Class”，旨在培养下一代AIGC人才，满足对AIGC领域高度专业人才的迫切需求。

师资力量

上海交通大学 AIGC Class 拥有多元化且卓越的师资力量，支持学生在AIGC领域的全面培养。师资队伍包括来自上海交通大学学生创新中心与人工智能研究院与的杰出教师，以及来自企业界的优秀工程师。

学生创新中心创新实践部现有全职教师15位，校内导师15位，校外专家30位，为交大学生提供全方位的“课-赛-创”教育内容服务。

人工智能研究院先共有百余名教师，视觉智能研究中心顺应人机物融合的发展趋势，瞄准人工智能和计算机视觉的科学前沿，发展以全域感知、行为认知、虚实融合、安全可信为特征的新一代视觉智能理论与技术，提升数字媒体、智慧城市、智慧医疗等领域的视觉感知认知能力，实现人工智能赋能新视界的愿景。

先进设备

光学动作捕捉系统：

动捕设备搭配了先进的Optitrack光学动捕系统，包括20台高精度Optitrack PrimeX22相机、多套动捕服装以及配套软件。让动捕演员穿上动捕服，在场地内随心所欲地运动，即可捕获完整的动作序列以供动画、游戏等使用。同时，场地内还搭设了多台三脚架以及RGB相机，用于视觉动捕等研究。

4D面部表情动作捕捉系统：

静态脸部PBR材质扫描仪包含46台三维建模相机与5台PBR高速相机，在3秒内拍摄46张建模图片与15x5张不同灯光条件下的PBR照片，用于重建高精度模型与8K分辨率的PBR材质贴图。

流体视觉直觉学习平台：

流体视觉直觉学习平台致力于研究多场景、多物理、多尺度耦合的人工智能流体仿真技术。通过高帧率高分辨率的相机阵列，实现对动态流体数据的多视角、高质量和同步性强的采集。

我们还有：

  • GPU阵列：华为Atlas 800、NVIDIA A100、NVIDIA RTX3090
  • 专属工作区与工作站
  • 各类AR/VR头盔

培养方案

每4人一组，针对不同项目的个性化培养方案，共1.5年。

工程能力培养（0.5年）

  • PBL项目式学习课程：
    • 计算机视觉与图形学
    • 机器学习与深度学习入门
    • 主流3D引擎使用
  • 工作坊：
    • 设备使用
    • 数据处理与数据分析

研究能力培养 （1年）

  • 工程创新：
    • 贡献开源项目：为未来的AIGC添砖加瓦
    • 参与顶级赛事：“挑战杯”与“互联网+”大赛
  • 学术创新：
    • 论文精读
    • 提出见解
    • 头脑风暴
    • 论文产出：在学术会议或期刊上发表文章
  • 导师制：
    • 每组将会分配一名科研或项目导师进行指导

结业

  • 每年我们会选拔优秀学员发放丰厚奖学金。
  • 成功结业将获得由学生创新中心和人工智能研究院联合颁发的结业证书。
  • 优秀学员将在人工智能研究院完成毕业设计，有机会进入人工智能研究院深造或赞助商企业实习。

招募要求

  • 上海交通大学本科二年级学生
  • 每届招募约20名
  • 需第一年成绩在年级前20%
  • SJTU AIGC Class 不收取学费，无学分
